# AI Phishing Detection System

## Overview

AI Phishing Detection System is a web application that analyzes email content and identifies potential phishing attempts. It calculates a threat score, classifies risk levels, and provides security recommendations.

## Features

* Email content analysis
* Threat score calculation
* Low, Medium, and High risk detection
* Security recommendations
* FastAPI backend
* Interactive web interface

## Tech Stack

* Python
* FastAPI
* HTML
* CSS
* JavaScript
* Git & GitHub

## Project Structure

```text
backend/
frontend/
README.md
requirements.txt
```

## Installation

```bash
pip install -r requirements.txt
python -m uvicorn backend.app:app --reload
```

## Usage

1. Start the backend server.
2. Open the frontend.
3. Enter email content.
4. Click Analyze.
5. View threat score and risk level.
